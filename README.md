# Portfólio - Daiani Oliveira

Site profissional de portfólio para **Massoterapia e Terapias Integrativas**, com foco em:

- atração de clientes
- acolhimento e profissionalismo
- agendamento rápido pelo WhatsApp

## Tecnologias

- HTML5
- CSS3
- JavaScript (vanilla)

## Estrutura

- `index.html`
- `style.css`
- `script.js`
- `assets/` (imagens)

## Funcionalidades

- layout responsivo (desktop e mobile)
- menu fixo
- animações suaves ao rolar
- botão de WhatsApp principal e flutuante
- cards de serviços com CTA de agendamento
- seção de contato com mapa e link "Como chegar"

## Como rodar localmente

Basta abrir o arquivo `index.html` no navegador.

## Publicação no GitHub Pages

1. Crie um novo repositório no GitHub (ex.: `portfolio-daiani`).
2. Envie os arquivos deste projeto para o repositório.
3. No GitHub, abra `Settings` > `Pages`.
4. Em `Source`, selecione:
   - `Deploy from a branch`
   - `Branch: main`
   - `Folder: / (root)`
5. Salve e aguarde a URL pública ser gerada.
